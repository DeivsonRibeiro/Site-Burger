# 🍔 Site Burger

## 📌 Sobre o Projeto

Landing page para uma hamburgueria, com foco em apresentação de produtos e design atrativo.

## 🛠️ Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## ⚙️ Funcionalidades

* Apresentação do cardápio
* Layout moderno
* Design responsivo

## 🎯 Objetivo

Praticar construção de interfaces visuais e desenvolvimento de páginas comerciais.

## 📚 Aprendizados

* Estilização com CSS
* Layout responsivo
* Estrutura de landing page

## ▶️ Como Executar

```bash
git clone https://github.com/DeivsonRibeiro/Site-Burger.git
cd Site-Burger
```

Abra o `index.html`.
